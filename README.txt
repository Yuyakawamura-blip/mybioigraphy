This package is prepared for static hosting on GitHub Pages.

Included:
- index.html
- assets/*.css
- assets/*.js
- favicon.svg
- og.png

Notes:
- The page markup and styling were captured from the live deployed site.
- The original bundled JavaScript files were protected by the host, so GitHub Pages-safe stub modules were added under the same filenames.
- The page remains viewable as a static site without relying on client-side hydration.
